# 📊 Integração RD Station → Google Sheets via Apps Script

## 📌 Visão Geral
Este projeto implementa uma integração entre o RD Station e o Google Sheets utilizando Google Apps Script.

O objetivo é automatizar o envio e registro de leads e dados de conversão diretamente em planilhas, permitindo análise, automação e criação de relatórios personalizados.

---

## 🎯 Objetivo
Centralizar dados de marketing e vendas em planilhas para:

- Monitoramento de leads em tempo real  
- Criação de dashboards personalizados  
- Integração com fluxos internos  
- Auditoria e histórico de dados  

---

## ⚙️ Funcionalidades
✔ Recebe dados do RD Station via webhook/API  
✔ Processa e formata os dados recebidos  
✔ Insere automaticamente os dados em planilhas Google  
✔ Permite personalização de colunas  
✔ Baixo custo e fácil manutenção
